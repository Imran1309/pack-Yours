# Pack Yours - Holiday Booking App

We introduce our website by using Gemini AI generated images.
Users can see places and book their stays and rooms.
The website is built using React.js and Tailwind CSS for styling, with images generated by Gemini AI.
The website is designed to help users book stays and rooms, making their trip unforgettable and memorable.
